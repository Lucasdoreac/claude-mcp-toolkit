## Overview

The Claude MCP Toolkit is a comprehensive CRM and business automation platform that helps manage deals, clients, and proposals efficiently.

### Features

- ✅ CRM Base
- ✅ Proposal Generator
- ✅ Email Integration
- ⏳ Web Dashboard
  - ✅ Frontend Base
  - ⏳ Backend API

### Tech Stack

Backend:
- FastAPI
- PostgreSQL
- Redis
- SQLAlchemy

Frontend:
- Next.js
- TypeScript
- Tailwind CSS
- React Query
